# Problem Statement:

Implement a program to find square of each element present in a matrix.

### Input Format

a 3x3 matrix

### Constraints

no

### Output Format

resultent matrix

### Sample Input 0
```
1 0 0
0 1 0
0 0 1
```
### Sample Output 0
```
1 0 0
0 1 0
0 0 1
```
### Sample Input 1
```
1 1 1
2 2 2
3 3 3
```
### Sample Output 1
```
1 1 1
4 4 4
9 9 9
```
