# Problem Statement:

Given a string, determine if it is a Palindrome string or not. A String is Palindrome if it is equal to reverse of the original string.

### Input Format

A String from the user

### Constraints

Non-empty String

### Output Format

Palindrome or not

### Sample Input 0
```
abba
```
### Sample Output 0
```
valid
```
### Sample Input 1
```
abc
```
### Sample Output 1
```
invalid
```
### Sample Input 2
```
madam
```
### Sample Output 2
```
valid
```
