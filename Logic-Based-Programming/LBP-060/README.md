# Problem Statement:

Given s string, implement a program to find max occurring character in the given string.

### Input Format

A string

### Constraints

No

### Output Format

max occuuring character

### Sample Input 0
```
welcome
```
### Sample Output 0
```
e
```
### Sample Input 1
```
java
```
### Sample Output 1
```
a
```
### Sample Input 2
```
aabbbccccddddd
```
### Sample Output 2
```
d
```
