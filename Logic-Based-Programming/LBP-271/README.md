# Problem Statement:

Geetha Singh is trying to create a system to convert binary number to its decimal equivalent. Help her to automate the process.

### Input Format

a binary value

### Constraints

no

### Output Format

decimal value

### Sample Input 0
```
1101
```
### Sample Output 0
```
13
```
### Sample Input 1
```
1111
```
### Sample Output 1
```
15
```
