# Problem Statement:

Implement a program to search for an element in an array.

### Input Format

size, array elements and element to search

### Constraints

size<100

### Output Format

search for the given element

### Sample Input 0
```
5
1 2 3 4 5
3
```
### Sample Output 0
```
2
```
### Sample Input 1
```
4
10 13 20 23
23
```
### Sample Output 1
```
3
```
