# Problem Statement:

Write a program to accept a number and check and display whether it is a Niven Number or not.<br>
__Niven Number is that a number which is divisible by its sum of digits.__

### Input Format

a number

### Constraints

n>0

### Output Format

Yes or No

### Sample Input 0
```
126
```
### Sample Output 0
```
Yes
```
### Sample Input 1
```
10
```
### Sample Output 1
```
Yes
```
