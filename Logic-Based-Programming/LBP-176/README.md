# Problem Statement:

Implement a program to read two integers values and return GCD of two numbers.

### Input Format

two space seperated integers.

### Constraints

no

### Output Format

GCD of two numbers

### Sample Input 0
```
98 56
```
### Sample Output 0
```
14
```
### Sample Input 1
```
2 3
```
### Sample Output 1
```
1
```
