# Problem Statement:

Write an algorithm to generate the token number from the application ID by doing this modifications.<br>
R1. If the digit is even add 1 to it.<br>
R2. If the digit is odd sub 1 from it.

### Input Format

a number from the user

### Constraints

no

### Output Format

token number

### Sample Input 0
```
2347
```
### Sample Output 0
```
3256
```
### Sample Input 1
```
245567
```
### Sample Output 1
```
354476
```
