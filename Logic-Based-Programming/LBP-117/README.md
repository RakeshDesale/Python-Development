# Problem Statement:

Implement a program to read array elements and find the second min element in an array.

### Input Format

size and array elements

### Constraints

size<100

### Output Format

return second min element

### Sample Input 0
```
5
1 8 7 5 2
```
### Sample Output 0
```
2
```
### Sample Input 1
```
7
10 11 12 7 8 9 -10
```
### Sample Output 1
```
7
```
### Sample Input 2
```
3
1 2 3
```
### Sample Output 2
```
2
```
