# Problem Statement:

Implement a program to check whether an array is paliandrome or not.

### Input Format

Array size N and Array Elements

### Constraints

no

### Output Format

true or false

### Sample Input 0
```
5
1 2 3 2 1
```
### Sample Output 0
```
true
```
### Sample Input 1
```
4
10 11 12 13
```
### Sample Output 1
```
false
```
