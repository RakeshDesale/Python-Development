# Problem Statement:

Implement a program to insert an element into an array at the first position

### Input Format

size,array elements and element to be inserted

### Constraints

size<100

### Output Format

return array after insertion

### Sample Input 0
```
5
1 2 3 4 5
10
```
### Sample Output 0
```
10 1 2 3 4 5
```
### Sample Input 1
```
4
7 8 9 10
5
```
### Sample Output 1
```
5 7 8 9 10
```
### Sample Input 2
```
3
2 3 4
1
```
### Sample Output 2
```
1 2 3 4
```
