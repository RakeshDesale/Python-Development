# Problem Statement:

Implement a program to read an array element and write on the screen.

### Input Format

size of the array and array elements

### Constraints

size<100

### Output Format

the given array

### Sample Input 0
```
5
1 2 3 4 5
```
### Sample Output 0
```
1 2 3 4 5
```
### Sample Input 1
```
8
1 2 3 8 5 2 7 3
```
### Sample Output 1
```
1 2 3 8 5 2 7 3
```
