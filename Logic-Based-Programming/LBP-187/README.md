# Problem Statement:

Given a string, create a new string made up of its last two letters, reversed and seperated by comma.

### Input Format

a string from the user

### Constraints

no

### Output Format

comma seperated last and second-last character

### Sample Input 0
```
demo
```
### Sample Output 0
```
o,m
```
### Sample Input 1
```
prakash
```
### Sample Output 1
```
h,s
```
