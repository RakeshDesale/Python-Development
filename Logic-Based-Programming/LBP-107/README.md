# Problem Statement:

Implement a program to read an array elements and print sum of all strong numbers in array.

### Input Format

size of the array and array elements

### Constraints

size<100

### Output Format

sum of all strong numbers

### Sample Input 0
```
4
1 2 145 40585
```
### Sample Output 0
```
40733
```
### Sample Input 1
```
5
1 2 3 4 5
```
### Sample Output 1
```
3
```
### Sample Input 2
```
1
145
```
### Sample Output 2
```
145
```
