# Problem Statement:

Implement a program to find the sum of elements available at odd index locations in an array.

### Input Format

size and array elements

### Constraints

no

### Output Format

print sum value

### Sample Input 0
```
5
1 2 3 4 5
```
### Sample Output 0
```
6
```
### Sample Input 1
```
5
1 1 1 1 1
```
### Sample Output 1
```
2
```
### Sample Input 2
```
6
1 2 3 4 5 6
```
### Sample Output 2
```
12
```
