# Problem Statement:

Create a function that replaces all the vowels in a string with a specified character,

### Input Format

A string from the user

### Constraints

No

### Output Format

A string

### Sample Input 0
```
welcome
#
```
### Sample Output 0
```
w#lc#m#
```
### Sample Input 1
```
prakash
.
```
### Sample Output 1
```
pr.k.sh
```
