# Problem Statement:

Two Strings a and b are called anagrams, if they contain all the same characters in the same frequencies

### Input Format

Two Strings

### Constraints

No

### Output Format

true or false

### Sample Input 0
```
anagram
margana
```
### Sample Output 0
```
true
```
### Sample Input 1
```
python
java
```
### Sample Output 1
```
false
```
