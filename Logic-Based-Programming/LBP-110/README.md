# Problem Statement:

Implement a program to sort the given array elements in ASC order.

### Input Format

size and array elements

### Constraints

size<100

### Output Format

sorted array in ASC

### Sample Input 0
```
4
1 3 2 4
```
### Sample Output 0
```
1 2 3 4
```
### Sample Input 1
```
5
5 1 3 2 4
```
### Sample Output 1
```
1 2 3 4 5
```
### Sample Input 2
```
6
10 50 20 40 30 60
```
### Sample Output 2
```
10 20 30 40 50 60
```
