# Problem Statement:

Implement a program to read array elements and find the min element in an array.

### Input Format

size and array elements.

### Constraints

size<100

### Output Format

return min element

### Sample Input 0
```
5
1 2 3 4 5
```
### Sample Output 0
```
1
```
### Sample Input 1
```
5
-1 -2 -3 -4 -5
```
### Sample Output 1
```
-5
```
### Sample Input 2
```
6
1 2 3 -1 -2 -3
```
### Sample Output 2
```
-3
```
### Sample Input 3
```
3
1 1 1
```
### Sample Output 3
```
1
```
