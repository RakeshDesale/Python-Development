# Problem Statement:

Given three numbers b,e, and m. Fill in a function that takes these three positive integer values and outputs b^e mod m.

### Input Format

b,e and m values

### Constraints

no

### Output Format

b^e mod m

### Sample Input 0
```
2 1 2
```
### Sample Output 0
```
0
```
### Sample Input 1
```
3 2 2
```
### Sample Output 1
```
1
```
