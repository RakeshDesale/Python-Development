# Problem Statement:

Given two positive numbers N and K, the task is to find the number N contains exactly K digit or not. If contains, then print Truedigit_count otherwise Falsedigit_count.

### Input Format

Value of N and K

### Constraints

no

### Output Format

True|False Digit_Count

### Sample Input 0
```
501 3
```
### Sample Output 0
```
True 3
```
### Sample Input 1
```
50121 4
```
### Sample Output 1
```
False 5
```
### Sample Input 2
```
505 5
```
### Sample Output 2
```
False 3
```
