# Problem Statement:
implement a program that takes a number as an argument, increments the number by +1 and returns the result

### Input Format

a number from the user

### Constraints

no constraints

### Output Format

an incremented value

### Sample Input 0
```
0
```
### Sample Output 0
```
1
```
### Sample Input 1
```
4
```
### Sample Output 1
```
5
```
