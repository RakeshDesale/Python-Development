# Problem Statement:

Given an integer N and integer array A as the input, where N denotes the length of A. Write a program to find an integer, the sum of all these product with successors.

### Input Format

array size and elements

### Constraints

no

### Output Format

an int value

### Sample Input 0
```
5
1 2 3 4 5
```
### Sample Output 0
```
70
```
### Sample Input 1
```
4
1 2 3 5
```
### Sample Output 1
```
50
```
