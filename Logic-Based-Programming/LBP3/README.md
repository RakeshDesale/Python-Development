# Problem Statement:
To check whether the given year is leap year or not.

### Input Format
year from the user

### Constraints
no constraint

### Output Format
<b>True</b> or <b>False</b>

### Sample Input 0
```
2020
```
### Sample Output 0
```
True
```
### Sample Input 1
```
2021
```
### Sample Output 1
```
False
```
