# Problem Statement:

Given a maximum of four digit to the base 17(10=>A,11=>B,12=>C,13=>D,14=>E,15=>F,16=>G) as input, output its decimal value.

### Input Format

a string value

### Constraints

no

### Output Format

an integer value with base 17

### Sample Input 0
```
D
```
### Sample Output 0
```
13
```
### Sample Input 1
```
ABC
```
### Sample Output 1
```
3089
```
### Sample Input 2
```
A
```
### Sample Output 2
```
10
```
