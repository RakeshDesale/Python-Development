# Problem Statement:

Create a function/method that takes a string and return the word count. The string will be a sentence.

### Input Format

A string

### Constraints

No

### Output Format

Word count

### Sample Input 0
```
This is test
```
### Sample Output 0
```
3
```
### Sample Input 1
```
Test Demo
```
### Sample Output 1
```
2
```
### Sample Input 2
```
Prakash
```
### Sample Output 2
```
1
```
