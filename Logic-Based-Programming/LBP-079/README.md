# Problem Statement:

Write a Java program to take an input string and exchange the first and last word and reverse the middle word.

### Input Format

a string

### Constraints

no

### Output Format

a string

### Sample Input 0
```
Hello welcome to java
```
### Sample Output 0
```
java ot emoclew Hello
```
### Sample Input 1
```
java is very easy
```
### Sample Output 1
```
easy yrev si java
```
