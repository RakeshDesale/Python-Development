# Problem Statement:

Accept video length in minutes the format is mm:ss in String format, create a function that takes video length and return it in seconds.

### Input Format

video length in mm:ss

### Constraints

no

### Output Format

length in seconds

### Sample Input 0
```
01:00
```
### Sample Output 0
```
60
```
### Sample Input 1
```
13:56
```
### Sample Output 1
```
836
```
### Sample Input 2
```
02:01
```
### Sample Output 2
```
121
```
