# Problem Statement:

Implement a program to check if a string contains only digits.

### Input Format

a string from the user

### Constraints

no

### Output Format

Yes or No

### Sample Input 0
```
abc123
```
### Sample Output 0
```
No
```
### Sample Input 1
```
123456
```
### Sample Output 1
```
Yes
```
### Sample Input 2
```
prakash22
```
### Sample Output 2
```
No
```
