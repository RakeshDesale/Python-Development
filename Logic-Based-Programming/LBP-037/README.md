# Problem Statement:

Implement a program to calculate toggle case of each characters of a string

### Input Format

A String from user

### Constraints

non-empty String

### Output Format

toggle case string

### Sample Input 0
```
Hello
```
### Sample Output 0
```
hELLO
```
### Sample Input 1
```
tEsT
```
### Sample Output 1
```
TeSt
```
### Sample Input 2
```
Java
```
### Sample Output 2
```
jAVA
```
