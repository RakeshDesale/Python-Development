# Problem Statement:

Write a program to find the digital root of a given number. Digital root is the recursive sum of digits in the given number (until single digit is arrived)

### Input Format

a number from the user

### Constraints

no

### Output Format

single digit number

### Sample Input 0
```
2478
```
### Sample Output 0
```
3
```
### Sample Input 1
```
69798
```
### Sample Output 1
```
3
```
