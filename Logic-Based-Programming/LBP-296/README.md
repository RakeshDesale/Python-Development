# Problem Statement:

Implement a program to print the list of integers which are divisible by 5 or 7.

### Input Format

a number from the user

### Constraints

no

### Output Format

seq of int values which are divisible by 5 or 7

### Sample Input 0
```
10
```
### Sample Output 0
```
5 7 10
```
### Sample Input 1
```
15
```
### Sample Output 1
```
5 7 10 14 15
```
