# Problem Statement:

Write a function that changes every letter to the next letter:<br>
"a" becomes "b"<br>
"b" becomes "c"<br>
"d" becomes "e"<br>
and so on ...

note: there is no z's in test cases, be happy.

### Input Format

a string from the user

### Constraints

no

### Output Format

modified string

### Sample Input 0
```
hello
```
### Sample Output 0
```
ifmmp
```
### Sample Input 1
```
bye
```
### Sample Output 1
```
czf
```
### Sample Input 2
```
welcome
```
### Sample Output 2
```
xfmdpnf
```
