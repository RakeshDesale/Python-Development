# Problem Statement:

Implement a program to find the area of the circle.

### Input Format

radius value

### Constraints

no

### Output Format

area of the circle (round to two decimals)

### Sample Input 0
```
12
```
### Sample Output 0
```
458.38
```
### Sample Input 1
```
1
```
### Sample Output 1
```
3.14
```
