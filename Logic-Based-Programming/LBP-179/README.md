# Problem Statement:

Given an array of size n, write a function to rearrange the numbers of the array in such that even and odd numbers are arranged alternatively in increasing order.

### Input Format

array size n and elements

### Constraints

no

### Output Format

updated array

### Sample Input 0
```
4
1 2 3 4
```
### Sample Output 0
```
2 1 4 3
```
### Sample Input 1
```
6
1 2 3 4 5 6
```
### Sample Output 1
```
2 1 4 3 6 5
```
