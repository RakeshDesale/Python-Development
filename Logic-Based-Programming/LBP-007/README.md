# Problem Statement:

Implement a program to extract digits from the given number

### Input Format

a number from the user

### Constraints

n>0

### Output Format

print digits in line sep by space

### Sample Input 0
```
123
```
### Sample Output 0
```
3 2 1
```
### Sample Input 1
```
145
```
### Sample Output 1
```
5 4 1
```
