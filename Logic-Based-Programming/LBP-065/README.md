# Problem Statement:

write a function that shutters a word as if someone is struggling to read it. The first two letters are repeated twice with an ellipsis ... , and then the word is pronounced with a question mark?

### Input Format

a string

### Constraints

no

### Output Format

xx... xx... ~~~~~~~?

### Sample Input 0
```
incredible
```
### Sample Output 0
```
in...in...incredible?
```
### Sample Input 1
```
abc
```
### Sample Output 1
```
ab...ab...abc?
```
