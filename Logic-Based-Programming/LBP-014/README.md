# Problem Statement:

Given an integer x, return x with its digits reversed.

### Input Format

a number from user

### Constraints

n>=0

### Output Format

reverse of the given number

### Sample Input 0
```
123
```
### Sample Output 0
```
321
```
### Sample Input 1
```
124
```
### Sample Output 1
```
421
```
### Sample Input 2
```
0
```
### Sample Output 2
```
0
```
