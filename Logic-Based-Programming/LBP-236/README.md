# Problem Statement:

Program to accept a matrix and check whether it is lower triangular matrix or not.

### Input Format

a 3x3 matrix

### Constraints

no

### Output Format

Yes or No

### Sample Input 0
```
1 0 0
4 5 0
7 8 9
```
### Sample Output 0
```
Yes
```
### Sample Input 1
```
1 0 0
0 1 0
0 0 1
```
### Sample Output 1
```
Yes
```
### Sample Input 2
```
1 2 3
4 5 6
7 8 9
```
### Sample Output 2
```
No
```
