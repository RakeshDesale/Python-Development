# Problem Statement:

Create a function that takes a string and returns true or false, depending on whether the characters are in order or not.

### Input Format

a string from the user

### Constraints

for non-empty string print invalid

### Output Format

true or false

### Sample Input 0
```
abc
```
### Sample Output 0
```
true
```
### Sample Input 1
```
prakash
```
### Sample Output 1
```
false
```
### Sample Input 2
```
123
```
### Sample Output 2
```
true
```
