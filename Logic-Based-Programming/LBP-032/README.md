# Problem Statement:

Program to count number of special characters and white spaces in a given string.

### Input Format

A string from the user

### Constraints

non-empty string

### Output Format

number of special characters

### Sample Input 0
```
wel123come@
```
### Sample Output 0
```
1
```
### Sample Input 1
```
abc@123#
```
### Sample Output 1
```
2
```
