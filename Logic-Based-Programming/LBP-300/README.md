# Problem Statement:

Implement a program to find sum of last three digits in the given number.

### Input Format

an int value

### Constraints

must be three digit number

### Output Format

int value

### Sample Input 0
```
2345
```
### Sample Output 0
```
12
```
### Sample Input 1
```
1234
```
### Sample Output 1
```
9
```
### Sample Input 2
```
1234
```
### Sample Output 2
```
9
```
