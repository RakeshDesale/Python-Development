# Problem Statement:

Implement a program to sort all the row wise elements in desc order.

### Input Format

a matrix

### Constraints

no

### Output Format

result matrix

### Sample Input 0
```
1 3 2
4 5 6
7 9 8
```
### Sample Output 0
```
3 2 1
6 5 4
9 8 7
```
### Sample Input 1
```
1 0 0
0 1 0
0 0 1
```
### Sample Output 1
```
1 0 0
1 0 0
1 0 0
```
