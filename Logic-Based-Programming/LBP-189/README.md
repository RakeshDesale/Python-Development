# Problem Statement:

Write a program to find the absolute difference between the original number and its reserved number.

### Input Format

a number from the user

### Constraints

no

### Output Format

absolute difference

### Sample Input 0
```
245
```
### Sample Output 0
```
297
```
### Sample Input 1
```
923
```
### Sample Output 1
```
594
```
### Sample Input 2
```
1234
```
### Sample Output 2
```
3087
```
