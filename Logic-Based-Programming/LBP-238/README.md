# Problem Statement:

Implement a program to read a matrix and check whether the given matrix is symmertric matrix or not.

### Input Format

a 3x3 matrix

### Constraints

no

### Output Format

Yes or No

### Sample Input 0
```
1 2 3
2 4 5
3 5 8
```
### Sample Output 0
```
Yes
```
### Sample Input 1
```
1 0 0
0 1 0
0 0 1
```
### Sample Output 1
```
Yes
```
