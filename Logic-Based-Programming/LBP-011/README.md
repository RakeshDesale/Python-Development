# Problem Statement:

Implement a program to calculate sum of prime digits present in the given number

### Input Format

a number from the user

### Constraints

n>0

### Output Format

print sum of prime digits

### Sample Input 0
```
123
```
### Sample Output 0
```
5
```
### Sample Input 1
```
101
```
### Sample Output 1
```
0
```
### Sample Input 2
```
142
```
### Sample Output 2
```
2
```
