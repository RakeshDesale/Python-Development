# Problem Statement:

Implement a program that takes a number as its argument and returns an array of all its factors.

### Input Format

a number

### Constraints

no

### Output Format

list of factors

### Sample Input 0
```
5
```
### Sample Output 0
```
1 5
```
### Sample Input 1
```
6
```
### Sample Output 1
```
1 2 3 6
```
### Sample Input 2
```
4
```
### Sample Output 2
```
1 2 4
```
