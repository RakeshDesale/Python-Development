# Problem Statement:

Implement a program to find number of even and odd elements in the given array

### Input Format

size and array elements

### Constraints

no

### Output Format

print number of even and odd elements line by line

### Sample Input 0
```
5
1 2 3 4 5
```
### Sample Output 0
```
2
3
```
### Sample Input 1
```
6
1 2 3 4 5 6
```
### Sample Output 1
```
3
3
```
### Sample Input 2
```
4
1 2 3 4
```
### Sample Output 2
```
2
2
```
### Sample Input 3
```
3
1 2 3
```
### Sample Output 3
```
1
2
```
