# Problem Statement:

Create a function which validates whether a given array alternates between positive and negative numbers.

### Input Format

an array size and array

### Constraints

no

### Output Format

true or false

### Sample Input 0
```
3
3 -2 5
```
### Sample Output 0
```
true
```
### Sample Input 1
```
6
3 -2 5 -5 2 -8
```
### Sample Output 1
```
true
```
