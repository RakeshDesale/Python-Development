# Problem Statement:

Implement a function/Method to return first character in each word from the given input string.

### Input Format

a string

### Constraints

no

### Output Format

first character in each string

### Sample Input 0
```
welcome to java programming
```
### Sample Output 0
```
wtjp
```
### Sample Input 1
```
Hi hello every one
```
### Sample Output 1
```
Hheo
```
