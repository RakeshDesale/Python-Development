# Problem Statement:

Implement a program to search for an element in an array.

### Input Format

size, array elements and element to search

### Constraints

size<100

### Output Format

search for the given element

### Sample Input 0
```
8
1 2 3 4 5 7 8 6
4
```
### Sample Output 0
```
3
```
### Sample Input 1
```
2
1 2
1
```
### Sample Output 1
```
0
```
### Sample Input 2
```
5
11 22 33 55 44
44
```
### Sample Output 2
```
3
```
