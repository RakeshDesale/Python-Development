# Problem Statement:

Implement a program that will test if a string is a valid PIN or not via a regular expression.<br>
A valid PIN has:

Exactly 4 characters.<br>
Only numeric characters (0-9).<br>
No whitespace.<br>

### Input Format

an input from the user

### Constraints

con

### Output Format

true or false

### Sample Input 0
```
0511
```
### Sample Output 0
```
true
```
### Sample Input 1
```
78945
```
### Sample Output 1
```
false
```
