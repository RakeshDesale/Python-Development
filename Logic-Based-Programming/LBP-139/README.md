# Problem Statement:

Implement a program to reverse only first half of the array and keep remaining elements as original.

### Input Format

size and array elements

### Constraints

no

### Output Format

reverse only first half of the array

### Sample Input 0
```
5
5 1 4 2 3
```
### Sample Output 0
```
1 5 4 2 3
```
### Sample Input 1
```
6
5 2 3 6 1 4
```
### Sample Output 1
```
2 3 5 6 1 4
```
