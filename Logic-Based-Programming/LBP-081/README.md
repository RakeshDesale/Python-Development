# Problem Statement:

Write a program to print even length words in a string?

### Input Format

a string from the user

### Constraints

no

### Output Format

list of strings with even length

### Sample Input 0
```
hello world java
```
### Sample Output 0
```
java
```
### Sample Input 1
```
python is very easy
```
### Sample Output 1
```
python is very easy
```
