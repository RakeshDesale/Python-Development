# Problem Statement:

Implement a program to find absolute diff between sum of max digits and sum of min digits present in three integers n1,n2 and n3.

### Input Format

n1,n2 and n3

### Constraints

no

### Output Format

int value

### Sample Input 0
```
5213
5698
2153
```
### Sample Output 0
```
12
```
### Sample Input 1
```
1212
1234
5678
```
### Sample Output 1
```
7
```
