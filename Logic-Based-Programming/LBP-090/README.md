# Problem Statement:

Given a string containing unique letters, return a sorted string with the letters that don't appear in the string.

### Input Format

a string from the user

### Constraints

no

### Output Format

return missing characters

### Sample Input 0
```
abcdefgpqrstuvwxyz
```
### Sample Output 0
```
hijklmno
```
### Sample Input 1
```
zyxwvutsrq
```
### Sample Output 1
```
abcdefghijklmnop
```
