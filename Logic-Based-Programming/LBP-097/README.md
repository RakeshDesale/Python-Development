# Problem Statement:

Implement a program to return number of consonants present in the given string

### Input Format

a string from the user

### Constraints

non-empty string

### Output Format

return number of consonants

### Sample Input 0
```
welcome
```
### Sample Output 0
```
4
```
### Sample Input 1
```
abc
```
### Sample Output 1
```
2
```
### Sample Input 2
```
xyz
```
### Sample Output 2
```
3
```
