# Problem Statement:

Implement a program to read an array elements and print sum of all prime elements.

### Input Format

size of the array and array elements

### Constraints

size<100

### Output Format

sum of all prime elements

### Sample Input 0
```
5
1 2 3 4 5
```
### Sample Output 0
```
10
```
### Sample Input 1
```
8
1 2 3 8 5 2 7 3
```
### Sample Output 1
```
22
```
### Sample Input 2
```
4
11 22 33 44
```
### Sample Output 2
```
11
```
