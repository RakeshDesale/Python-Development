# Problem Statement:

Implement a program to update every array element with multiplication of previous and next numbers in array.

### Input Format

size and array elements

### Constraints

no

### Output Format

updated array

### Sample Input 0
```
6
1 2 3 4 5 6
```
### Sample Output 0
```
2 3 8 15 24 5
```
### Sample Input 1
```
4
10 20 30 40
```
### Sample Output 1
```
20 300 800 30
```
