# Problem Statement:

Implement a program to display next 5 characters after input character.

### Input Format

a character from the user

### Constraints

no

### Output Format

next 5 characters exluding input

### Sample Input 0
```
a
```
### Sample Output 0
```
b c d e f
```
### Sample Input 1
```
m
```
### Sample Output 1
```
n o p q r
```
