# Problem Statement:

Implement a program to find the number of occurrences of the given element.

### Input Format

size,array element and key element

### Constraints

size<100

### Output Format

return number of occurrences

### Sample Input 0
```
5
1 2 3 4 5
3
```
### Sample Output 0
```
1
```
### Sample Input 1
```
8
1 1 1 1 1 1 1 8
1
```
### Sample Output 1
```
7
```
### Sample Input 2
```
8
1 1 1 1 1 1 1 8
8
```
### Sample Output 2
```
1
```
