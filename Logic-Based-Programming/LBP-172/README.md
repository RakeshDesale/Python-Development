# Problem Statement:

Given a maximum of 100 digit numbers as input, find the difference between the sum of odd and even position digits.

### Input Format

a number from the user

### Constraints

no

### Output Format

an integer

### Sample Input 0
```
12
```
### Sample Output 0
```
1
```
### Sample Input 1
```
123
```
### Sample Output 1
```
-2
```
