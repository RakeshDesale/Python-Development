# Problem Statement:

Implement a program to delete the given element from an array

### Input Format

size,array elements and element

### Constraints

size<100

### Output Format

return array after deleting

### Sample Input 0
```
5
7 6 1 2 3
1
```
### Sample Output 0
```
7 6 2 3
```
### Sample Input 1
```
3
11 22 33
44
```
### Sample Output 1
```
-1
```
### Sample Input 2
```
4
1 2 3 4
3
```
### Sample Output 2
```
1 2 4
```
