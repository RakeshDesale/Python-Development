# Problem Statement:

Implement a program to find sum of even number between x and y both are inclusive.

### Input Format

two int values

### Constraints

no

### Output Format

sum of even numbers between x and y

### Sample Input 0
```
10
15
```
### Sample Output 0
```
36
```
### Sample Input 1
```
0
10
```
### Sample Output 1
```
30
```
