# Problem Statement:

Implement a program to create an array with n elements by taking multiples of m.

### Input Format

m and n

### Constraints

size of the array must be n

### Output Format

return an array with n elements which contains multiples of m.

### Sample Input 0
```
7
5
```
### Sample Output 0
```
7 14 21 28 35
```
### Sample Input 1
```
12
10
```
### Sample Output 1
```
12 24 36 48 60 72 84 96 108 120
```
### Sample Input 2
```
2
5
```
### Sample Output 2
```
2 4 6 8 10
```
